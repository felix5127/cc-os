# Performance

## Model Selection Strategy

| Model | When to Use |
|-------|-------------|
| **Haiku** | Lightweight agents, pair programming, worker agents |
| **Sonnet** | Main development, orchestration, complex coding |
| **Opus** | Architectural decisions, deep reasoning, research |

## Context Window Management

High context sensitivity (avoid last 20%):
- Large-scale refactoring
- Multi-file feature implementation
- Complex debugging

Low context sensitivity:
- Single-file edits
- Utility creation
- Documentation
